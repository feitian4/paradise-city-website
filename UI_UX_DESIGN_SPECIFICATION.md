# 第一部分：设计系统基础

## 1.1 设计哲学

### 核心理念
> "数字禅意 · 琉璃光界" —— 将藏传佛教的庄严神圣与现代数字美学完美融合

### 设计关键词
- **神圣** (Sacred) —— 传达宗教的庄严与敬畏
- **宁静** (Serene) —— 营造心灵的平和与安宁
- **通透** (Luminous) —— 体现琉璃光的纯净与明亮
- **现代** (Modern) —— 展现Web3科技的前沿感

## 1.2 色彩系统

### 主色调
| 色彩名称 | HEX值 | RGB值 | 使用场景 |
|---------|-------|-------|---------|
| **深空黑** | `#0A0A0F` | rgb(10,10,15) | 主背景、深色区块 |
| **藏金** | `#D4AF37` | rgb(212,175,55) | 主强调色、按钮、标题 |
| **琉璃蓝** | `#1E3A5F` | rgb(30,58,95) | 次级背景、卡片 |
| **净白** | `#F8F6F0` | rgb(248,246,240) | 文字主色、浅色背景 |

### 辅助色
| 色彩名称 | HEX值 | 使用场景 |
|---------|-------|---------|
| **朱红** | `#C41E3A` | 重要提醒、藏式装饰 |
| **松石绿** | `#0D7377` | 藏药相关、自然元素 |
| **琥珀** | `#FFBF00` | 高亮、悬停状态 |
| **紫罗兰** | `#7B68EE` | NFT稀有度标识 |

### 渐变色
```css
/* 琉璃光渐变 - 主视觉 */
--gradient-lapis: linear-gradient(135deg, #0A0A0F 0%, #1E3A5F 50%, #D4AF37 100%);

/* 金色辉光 - 按钮/强调 */
--gradient-gold: linear-gradient(90deg, #B8860B 0%, #D4AF37 50%, #FFD700 100%);

/* 深邃空间 - 元宇宙背景 */
--gradient-cosmos: radial-gradient(ellipse at center, #1E3A5F 0%, #0A0A0F 70%);
```

## 1.3 字体系统

### 中文字体
- **标题字体**：思源宋体 (Source Han Serif CN) - 传达古典庄重
- **正文字体**：思源黑体 (Source Han Sans CN) - 确保阅读舒适
- **藏文显示**：Noto Sans Tibetan - 支持藏文经文展示

### 英文字体
- **Display**：Playfair Display - 优雅衬线，用于大标题
- **Body**：Inter - 现代无衬线，用于正文
- **Code**：JetBrains Mono - 代码展示

### 字体层级
| 层级 | 字号 | 字重 | 行高 | 用途 |
|-----|------|-----|------|-----|
| H1 | 48px/3rem | 700 | 1.2 | 页面主标题 |
| H2 | 36px/2.25rem | 600 | 1.3 | 板块标题 |
| H3 | 24px/1.5rem | 600 | 1.4 | 卡片标题 |
| H4 | 20px/1.25rem | 500 | 1.5 | 小标题 |
| Body | 16px/1rem | 400 | 1.6 | 正文内容 |
| Caption | 14px/0.875rem | 400 | 1.5 | 辅助说明 |
| Small | 12px/0.75rem | 400 | 1.4 | 标签/注释 |

## 1.4 间距系统

### 基础单位：8px
```css
--space-xs: 4px;
--space-sm: 8px;
--space-md: 16px;
--space-lg: 24px;
--space-xl: 32px;
--space-2xl: 48px;
--space-3xl: 64px;
--space-4xl: 96px;
```

### 区块间距
- **Section Padding**: 80px-120px (桌面) / 48px-64px (移动端)
- **Container Max Width**: 1440px
- **Grid Gap**: 24px-32px

## 1.5 圆角与阴影

### 圆角系统
```css
--radius-sm: 4px;    /* 按钮、标签 */
--radius-md: 8px;    /* 卡片、输入框 */
--radius-lg: 16px;   /* 大卡片、弹窗 */
--radius-xl: 24px;   /* 特色卡片 */
--radius-full: 9999px; /* 圆形、头像 */
```

### 阴影系统
```css
/* 微妙阴影 - 卡片默认 */
--shadow-sm: 0 2px 8px rgba(0,0,0,0.2);

/* 中等阴影 - 悬停状态 */
--shadow-md: 0 4px 16px rgba(0,0,0,0.3);

/* 强烈阴影 - 弹窗/浮层 */
--shadow-lg: 0 8px 32px rgba(0,0,0,0.4);

/* 金色辉光 - 强调元素 */
--shadow-gold: 0 0 20px rgba(212,175,55,0.3);

/* 琉璃光晕 - 神圣元素 */
--shadow-lapis: 0 0 30px rgba(30,58,95,0.5);
```

## 2.2 首页设计

### 首屏 (Hero Section)
**布局**：全屏沉浸式视频背景 + 居中标题

**元素构成**：
- **背景**：4K循环视频 - 香格里拉日出、经幡飘动、玛尼石堆
- **遮罩**：深色渐变遮罩 (opacity 0.6)
- **主标题**：
  ```
  香格里拉天堂之城
  Paradise City of Shangri-La
  གནས་མཆོག་གྲོང་ཁྱེར།
  ```
  字体：思源宋体 72px + Playfair Display 48px + 藏文 36px
  动效：逐字淡入，总时长2秒

- **副标题**："通往内心宁静之门 · Your Portal to Inner Peace"
- **CTA按钮组**：
  - 主按钮："开启疗愈之旅" (金色渐变)
  - 次按钮："探索元宇宙" (描边样式)

- **七佛图标栏**：底部横向排列7个佛像剪影，点击可跳转

**交互动效**：
- 鼠标移动：背景视差效果 (parallax)
- 滚动：标题渐隐，导航栏固定并改变透明度
- 七佛图标：悬停时放大1.1倍，显示佛名tooltip

### 七佛愿力板块
**布局**：左右交替图文排版

**每个佛像展示包含**：
- **左侧**：佛像插画 (SVG动画) 或 3D预览
- **右侧**：
  - 佛名 (H2)
  - 梵文/藏文名称
  - 愿力描述 (引用经文)
  - 现实疗愈映射
  - "深入了解" 按钮

**视觉特色**：
- 每个佛像配色不同，对应七佛各自的能量色彩
- 滚动触发：佛像SVG路径动画绘制
- 背景：微妙的曼陀罗图案

### 药师定修行板块
**布局**：三列卡片式

**三张主卡片**：
1. **色法 - 大食法**
   - 图标：餐盘/草药
   - 标题：藏医药食疗
   - 描述：通过藏医药调理身体
   
2. **息法 - 安般法**
   - 图标：呼吸/莲花
   - 标题：呼吸冥想
   - 描述：高原氧疗与呼吸修行
   
3. **心法 - 入寤法**
   - 图标：月亮/梦境
   - 标题：梦境修行
   - 描述：睡眠疗愈与潜意识探索

**卡片样式**：
- 背景：`#1E3A5F` 带纹理
- 边框：1px solid `#D4AF37` (悬停时发光)
- 圆角：16px
- 悬停：上浮8px + 金色阴影

### 数字净土板块
**布局**：画廊式展示

**NFT展示区**：

- 每张卡片：NFT预览图 + 名称 + 稀有度标签
- 悬停：3D翻转效果显示详情

**元宇宙入口**：
- 大面积视觉区：七层世界预览图
- "进入净琉璃光界" 按钮
- 实时在线人数显示

### 实修基地板块

**交互地图**：
